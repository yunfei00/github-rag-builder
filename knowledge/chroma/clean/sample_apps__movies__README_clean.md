# Movies with Chroma

Setup your `.env` file:

```
OPENAI_API_KEY=
CHROMA_API_KEY=
CHROMA_TENANT=
CHROMA_DATABASE=
CHROMA_HOST=https://api.trychroma.com
```

Then, run the development server:

```bash
npm install
npm run dev
```
